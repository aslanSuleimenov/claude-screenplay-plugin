# Проект: Claude Screenplay Plugin — плагин для написания сценариев

## Что это
Плагин для Claude Code для написания сценариев (художественных и документальных). 17 slash-команд, 5 агентов, хуки (PostToolUse, SessionStart, Stop), система компаса с 26 жанрами художественного кино + 7 типами документального кино, конвертер MD→DOCX.

## Структура
```
commands/          — 17 команд
agents/            — 5 агентов
hooks/             — хуки
compass/           — fiction/ + doc/ (жанровые контракты)
converter_MD_DOCX/ — конвертер
analytics/         — avoid-ai-writing-tells.md
```
